# ☕ Caff Own Coffee — Aesthetic Cafe Website

A modern, multi-page aesthetic cafe website designed for today's generation.  
Built with a clean UI, smooth animations, and a warm orange-beige theme to reflect a cozy cafe vibe.

---

## 🌐 Live Demo
🔗 https://nishchay-mn.github.io/caff-own-coffee

---

## 📌 Project Overview
This project is a fully responsive **frontend website** created for a cafe to showcase:
- Menu items 🍝  
- Cafe ambiance ✨  
- Customer reviews ⭐  
- Contact details 📍  

The goal was to create a **modern, visually appealing website** that attracts young customers and improves online presence.

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the website  
- **CSS3** → Styling, layout, animations  
- **JavaScript (Vanilla JS)** → Interactivity and effects  

---

## 🎨 Features
- 🔥 Multi-page layout (Home, About, Menu, Gallery, Contact)  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🎞️ Smooth scroll animations  
- 🖱️ Custom cursor effect  
- 🌌 Animated hero section  
- 🃏 Interactive menu cards  
- 📊 Count-up statistics (ratings, reviews)  
- 📌 Google Maps location section  
- 🍽️ Menu filter system (Drinks / Snacks / Mains)  

---

## 🎯 Theme & Design
- **Primary Color:** #EA2E00 (Burnt Orange)  
- **Secondary Color:** #F0E7D6 (Beige)  

Inspired by modern aesthetic cafes with a minimal and elegant UI.

## 🚀 How to Run Locally
1. Download or clone this repository  
2. Open the project folder  
3. Double-click `index.html`  
4. Website will open in your browser

## 📤 Deployment
This project is deployed using **GitHub Pages**:
- Push code to GitHub repository  
- Enable Pages in settings  
- Get live URL instantly 🌐
